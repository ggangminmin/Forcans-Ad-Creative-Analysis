# 🚀 AI Ad Creative Agent POC 가이드

본 POC는 시장 트렌드 분석부터 광고 카피 생성, 시각화 시안 설계까지 세 명의 전문 에이전트가 협업하는 **통합 광고 기획 파이프라인**입니다.

---

## 🏛️ 에이전트 기술 구성 (Agent Context)

모든 작업은 최신 AI 모델과 특화된 프롬프트 지침(`.agents` 폴더)을 기반으로 수행됩니다.

| 워크플로우 단계 | 담당 에이전트 | 주요 역할 | 산출물 |
| :--- | :--- | :--- | :--- |
| **1. 트렌드 분석** | `scouter.md` | 시장 트렌드 및 광고 피로도 분석 | 핵심 인사이트, 키워드, 매체 컨텍스트 |
| **2. 소재 제안** | `writer.md` | 감성/유머/대담 등 톤별 카피 창작 | 광고 카피 12안, A/B CTR 예측 데이터 |
| **3. 비주얼 설계** | `designer.md` | 컷별 구도 및 시각화 프롬프트 설계 | 9컷 비주얼 시안, 리얼타임 이미지 생성 |

---

## 🕹️ 단계별 사용 방법 (Workflow)

### 1단계: 분석 조건 설정
1.  왼쪽 **사이드바**에서 타겟 제품의 **카테고리** 입력
2.  **매체 포맷**(인스타 스토리, 유튜브 쇼츠 등) 및 **광고 톤** 선택
3.  **[🔍 트렌드 분석]** 버튼 클릭
    *   **결과**: 시장 핵심 인사이트와 소비자 피로 문구(사용 금지어)가 출력됩니다.

### 2단계: 광고 소재 및 서사 설계
1.  분석 결과 카드 우측의 **[소재 제안 받기 →]** 버튼 클릭
2.  생성된 12가지 카피 중 최적의 소재 선택
    *   **A/B CTR 예측**: 상단의 버튼을 통해 타겟별 예상 클릭률 시뮬레이션 확인 가능
3.  마음에 드는 카드의 **[비주얼 제작 →]** 버튼 클릭

### 3단계: 9컷 비주얼 제작 및 피드백
1.  9컷 그리드 레이아웃과 컷별 연출 가이드 확인
2.  각 컷을 **클릭**하여 AI 실시간 이미지 생성 (Nano Banana 연동)
3.  **수정 버튼**: 개별 컷에 대한 텍스트 피드백을 입력하여 즉시 재생성 가능

---

## 🛠️ 핵심 기능 및 특징
*   **통합 사이드바 워크플로우**: 모든 제어와 단계 이동을 왼쪽 패널에서 직관적으로 관리합니다.
*   **매체 최적화**: 9:16, 1:1 등 매체별 규격에 맞춘 비주얼 기획을 제안합니다.
*   **실시간 피드백 루프**: 기획자가 의도한 대로 시안을 즉석에서 수정하고 업데이트할 수 있습니다.
*   **에셋 다운로드**: 기획된 모든 텍스트와 프롬프트 데이터를 JSON 형식으로 즉시 다운로드 가능합니다.
